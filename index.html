Grader's Notes:
This is a Three.js scene I made of a small fox stargazing! Meant to be a snowy scene with randomly placed trees, foliage, and logs, as well as snow drifts.

Notable Mentions:
- Followed a tutorial to add snow particle effects
- Extended it to add a Star to the scene as a Point Geometry; Star image was made using Clip Studio Paint
- Further added smaller stars to the scene to create a starry sky
- Made the rock texture on the 3D primitives using Procreate
- Added some post-processing effects, i.e. the bloom
